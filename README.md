# BTBT / WYFI NAV Dashboard

Live dashboard:
https://1bsar.github.io/BTBT-WYFI-NAV/

Interactive NAV and valuation model for Bit Digital (BTBT) and WhiteFiber (WYFI).

The model estimates BTBT implied value under different:

* WYFI price scenarios,
* ETH treasury valuations,
* and holding-company discount assumptions.

Includes:

* scenario analysis,
* historical discount calibration,
* NAV sensitivity modeling,
* and valuation waterfall visualization.

Created as part of an independent pre-earnings research thesis and ongoing sector analysis.

Not financial advice.
