# BTBT / WYFI NAV Dashboard

Live dashboard: https://1bsar.github.io/BTBT-WYFI-NAV/

This project is an interactive NAV model for Bit Digital (BTBT) and WhiteFiber (WYFI). It estimates BTBT’s implied net asset value using WYFI equity value, ETH treasury value, cash, debt, and holding-company discount assumptions.

The dashboard includes:
- NAV sensitivity by WYFI and ETH price
- holding-company discount scenarios
- historical discount calibration
- waterfall analysis
- scenario table

Built with HTML, CSS, JavaScript, and Chart.js.

Not financial advice.
